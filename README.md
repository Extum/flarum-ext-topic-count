## Installation

Use [Bazaar](https://discuss.flarum.org/d/5151-flagrow-bazaar-the-extension-marketplace) or install manually:

```bash
composer require extum/flarum-ext-topic-count
```

## Updating

```bash
composer require extum/flarum-ext-topic-count
php flarum cache:clear
```
